Thank you for playing Monster Hunter Now!

This article contains the main release information for V113 released on 2026/1/13 JST. Please note that not all updates are covered.

Last Update: 2026/1/13

Adjustments/Changes
General Hunting
Certain weapon types have been adjusted to make them more likely to inflict abnormal status ailments, taking into account the overall balance of abnormal status ailments across all weapon types.
The affected weapons are as follows:

All ailments:
Sword & shield/great sword/long sword/hunting horn/lance/switch axe/bow
Poison/Sleep/Blast:
Dual blades
Blast:
Gunlance
Revisions made to the UI of the hunt prep screen when accessing from the Paintball List.

When inviting friends to group hunts via QR code, the QR code will now automatically appear on the lobby screen.
Adjustments made to the switch axe’s Special Skill when used on Garangolm’s head, as the number of hits inflicted mid-attack was inconsistent.

Driftsmelting
When selecting equipment for driftsmelting, up to 8 of your most recently driftsmelted equipment will now be displayed.
Equipment
In order to make loadouts easier to use, filters and a lock function have been added to the loadout screen.

Filters

Weapon types and attribute icons added, sorting now possible.
Tapping an icon will display loadouts that contain the relevant equipment.
Button to swap between weapon type/element filters added.
Lock function
In order to prevent unintended changes to loadouts, a lock function has been added.

Lock button (shown as lock icon) added to loadouts.

Tap the lock button to toggle the lock on or off.

While locked, the following actions cannot be performed on loadouts:

Renaming
Changes to set weapon/armor
*In order to make changes, please remove the lock first.
Note: Even when locked, skills assigned to armor driftsmelting slots can still be changed.

Loadouts unlocked via the Extreme Hunter Pass cannot have their lock status toggled once the pass expires.

Adjustments made to the visual size of light bowgun weapons when not equipped.

The weapon size will have no effect on hunting.
Upcoming Update
As announced in the 4th Now Navi, we plan to distribute an Item Box Expansion (+250) to all players who log in after the distribution start date, targeting late January 2026. Please look forward to it.
